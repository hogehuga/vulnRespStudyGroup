# Marp Theme: Formal

ビジネス、特にセキュリティ関連のプレゼンテーション向けにデザインされた Marp for VS Code 用のカスタムテーマです。
TLP (Traffic Light Protocol) の表示切り替え、セクション区切り、会社ロゴの配置などに標準対応しています。

## 基本設定

サンプルとして同じディレクトリに、`sample.md` と `sample.pdf` を配置しています。
MarpのMarkdownファイルの先頭で以下のように指定してください。

```
---
marp: true
theme: formal
---
```

## 機能と使い方

### 1. ロゴの表示

デフォルトではロゴは表示されません。 ロゴを表示したい場合、Markdown内に `<style>` タグを記述し、ローカル画像のパスを指定してください。

```
<style>
:root {
  /* プレゼンファイルからの相対パスで指定 */
  --logo-image: url('./assets/logo.png');
}
</style>
```

### 2. TLP(Traffic Light Protocol)の切替え

全ページ共通で左上にTLPラベルが表示されます。 デフォルトは TLP:GREEN です。ページごとに変更したい場合は、_class ディレクティブを使用します。

|クラス名|色|意味|
|:--|:--:|:--|
|tlp-white|緑|Disclosure is not limited|
|tlp-green|白|(Default)Basic info|
|tlp-amber|黄|Limited disclosure|
|tlp-red|赤|Not for disclosure|
|tlp-none|なし|TLPを表示しない|

- TLPは、First TLP2.0を参照
  - https://www.first.org/tlp/
- TPLの画像は、本githubのmarp-themes/TLP-*.png を利用

```
---

# デフォルトのスライド

このスライドは、TLP-GREENが適用されます。

---

# TLP-REDのスライド

<!-- _class: tlp-red -->

このスライドは、TLP-REDが適用されます。

---

# 他のclassと併用する

<!--  _class: title red-->

このスライドは、titleスライド且つTLP-REDが適用されます。

```

### 3. スライドの種類 (レイアウト)

状況に合わせて以下のクラスを使い分けてください。

|用途|class|
|:--|:--|
|タイトルスライド|title|
|セクション区切り/扉|lead|
|通常スライド|(指定なし)|
|反転カラー|invert|
|本文小さい文字|small|

#### a. title

プレゼンテーションの表紙で使います。

|セクション|概要|
|:--|:--|
|H1|タイトルを（中央）|
|H2|サブタイトル（中央、タイトル下）|
|本文|発表者情報等（右下）|

```
---
<!-- _class: title -->

# 脆弱性管理の現状と課題
## 2025年度上期報告会

hogehuga
脆弱性管理推進課
有限会社 脆弱性対応研究会
2025-11-25

---
```

#### b. lead

章の変わり目などに使用します。

|セクション|概要|
|:--|:--|
|H1|セクションタイトル（上部固定）|
|H2|概要説明（H1下部）|
|H3|所要時間など（左上）|
|リスト|アジェンダ（右下）|

リストで全体を示し、今はどの賞を話しているかを `**` で強調する想定です。

```
---
<!-- _class: lead -->

# 1. 本講演について
## 本講演の概要と全体アジェンダ
### 約5分

- **1.本講演について**
- 2.弊社の脆弱性管理の現状
- 3.課題と対応
- 4.下期のアクション

---
```

#### c. (指定なし)

一般的なコンテンツを表示します。

|セクション|概要|
|:--|:--|
|h1|スライドタイトル|
|h2|スライド内での大見出し（グレー）|
|h3|スライド内での中見出し（グレー、H2より少し小さい）|

```
---
# 現状の管理体制

## 組織構成

情報システム部と連携し、以下のような役割を担っている。

### 情報システム部

- NW担当：クラウド全体管理とオンプレミスネットワーク全体管理
- セキュリティ担当：全アセットのセキュリティ設定を包括管理
- ...

---
```


#### d. invert

黒ベース（ダークグレー）の背景に切り替わります。 重要な警告や、話題転換のアクセントに使用します。
通常スライドと同じ動作になります。

```
---
<!-- _class: invert -->

# 組織的な問題で対応が遅れてしまう

以下のような組織に関わる事象で、対応が遅れる

- 対応基準が存在していない
- 対応に関する経営層承認に時間がかかっている
- ...

---
```

#### e. small

情報量が多いスライドのために、本文の文字サイズを標準の 45px から 40px に縮小します。 他のクラス（TLP変更など）と併用可能です。
構成は通常スライドと同じです。

```
---
<!-- _class: small-->

# 詳細な説明

表示される文字サイズが減少することにより、ログやより詳細な説明を記載できる。
プレゼンテーションは、全体を理解するために文字を少なく要点を記載したもの、より深く理解するために詳細な説明を行うもの、と複数の用途があります。本classは、詳細な説明用途を想定しています。しかしながらsmallを多用すると聴講者には情報過多となるため、プレゼンテーション全体としてはあまり望ましくありません。

---
```
