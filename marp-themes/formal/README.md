# Marp Theme: Formal

ビジネス、特にセキュリティ関連のプレゼンテーション向けにデザインされた Marp for VS Code 用のカスタムテーマです。
TLP (Traffic Light Protocol) の表示切り替え、セクション区切り、会社ロゴの配置などに標準対応しています。

## 基本設定

サンプルとして同じディレクトリに `sample.md` と `sample.pdf` を配置しています。
MarpのMarkdownファイルの先頭で以下のように指定してください。

```markdown
---
marp: true
theme: formal
---
```

## 機能と使い方

### 1. ロゴの表示

デフォルトではロゴは表示されません。ロゴを表示したい場合、Markdown内に `<style>` タグを記述し、ローカル画像のパスを指定してください。

```markdown
<style>
:root {
  /* プレゼンファイルからの相対パスで指定 */
  --logo-image: url('./assets/logo.png');
}
</style>
```

ロゴは全スライドの右上に最前面 (z-index: 103) で表示されます。ヘッダー帯やコンテンツより上に描画されます。

### 2. TLP (Traffic Light Protocol) の切替え

TLP 2.0 に準拠したラベルを左上に表示します。色定義および意味は [FIRST TLP 2.0](https://www.first.org/tlp/) を参照してください。

| クラス名    | 背景        | 文字色          | 用途                                                    |
| :---------- | :---------- | :-------------- | :------------------------------------------------------ |
| `tlp-red`   | `#000000`   | `#FF2B2B`       | Not for disclosure, restricted to participants only     |
| `tlp-amber` | `#000000`   | `#FFC000`       | Limited disclosure, restricted to participants' organization |
| `tlp-green` | `#000000`   | `#33FF00`       | Limited disclosure, restricted to the community         |
| `tlp-clear` | `#000000`   | `#FFFFFF`       | Recipients can spread this to the world                 |
| `tlp-none`  | —           | —               | TLPラベルを表示しない (デフォルト)                       |

TLPラベルは外部画像を参照せず、CSSにSVG data URLとして埋め込まれています。インターネット接続のない環境でも正しくレンダリングされます。

#### TLPの指定方法

デフォルトではTLPラベルは表示されません。Marpの `class` / `_class` ディレクティブで切り替えます。

- `<!-- class: X -->`: それ以降の全スライドに適用 (アンダースコアなし)
- `<!-- _class: X -->`: そのスライドのみに適用 (アンダースコアあり)

```markdown
---
marp: true
theme: formal
---

<!-- class: tlp-green -->

# デフォルトのスライド

このスライドは `tlp-green` が適用されます (以降のスライドも継続適用)。

---

# 別の通常スライド

`tlp-green` が継続して適用されます。

---

<!-- _class: tlp-red -->

# このスライドだけ TLP:RED

`_class` を使うとそのスライドのみに適用されます。次のスライドからは `tlp-green` に戻ります。

---

<!-- _class: title tlp-red -->

# 他クラスとの併用

スペース区切りで複数指定すると、`title` レイアウト且つ TLP:RED が適用されます。
```

### 3. スライドの種類 (レイアウト)

状況に合わせて以下のクラスを使い分けてください。

| 用途                | クラス       |
| :------------------ | :----------- |
| タイトルスライド     | `title`      |
| セクション区切り/扉 | `lead`       |
| 通常スライド         | (指定なし)   |
| 反転カラー           | `invert`     |

#### a. `title` — タイトルスライド

プレゼンテーションの表紙に使います。

| Markdown要素        | 表示位置・スタイル                        |
| :------------------ | :---------------------------------------- |
| `#` (H1)            | タイトル (中央, 100px, teal下線つき)       |
| `##` (H2)           | サブタイトル (中央, タイトル下, グレー)    |
| 本文 (段落)         | 発表者情報など (右下固定)                  |

```markdown
---

<!-- _class: title -->

# 脆弱性管理の現状と課題
## 2025年度上期報告会

hogehuga
脆弱性管理推進課
有限会社 脆弱性対応研究会
2025-11-25

---
```

#### b. `lead` — セクション区切り

章の変わり目などに使用します。アジェンダリストで全体構造を示し、現在話している章を `**強調**` で示す想定です。

| Markdown要素        | 表示位置・スタイル                        |
| :------------------ | :---------------------------------------- |
| `#` (H1)            | セクションタイトル (左上, 90px, teal下線) |
| `##` (H2)           | 概要説明 (H1の下, 45px, 通常ウェイト)     |
| `###` (H3)          | 所要時間など (左上ヘッダー下, グレー)     |
| 箇条書き (ul)        | アジェンダ (右下固定)                      |
| 箇条書き内の `**…**` | 現在地強調 (teal, 拡大表示)               |

```markdown
---

<!-- _class: lead -->

# 1. 本講演について
## 本講演の概要と全体アジェンダ
### 約5分

- **1. 本講演について**
- 2. 弊社の脆弱性管理の現状
- 3. 課題と対応
- 4. 下期のアクション

---
```

#### c. (指定なし) — 通常スライド

一般的なコンテンツを表示します。

| Markdown要素        | 表示位置・スタイル                              |
| :------------------ | :---------------------------------------------- |
| `#` (H1)            | スライドタイトル (左寄せ, 70px, teal下線つき)   |
| `##` (H2)           | スライド内の大見出し (teal, 細い下線つき)        |
| `###` (H3)          | スライド内の中見出し (グレー, 下線なし)          |
| 本文 (段落, リスト)  | 通常テキスト (40px)                              |

```markdown
---

# 現状の管理体制

## 組織構成

情報システム部と連携し、以下のような役割を担っている。

### 情報システム部

- NW担当: クラウド全体管理とオンプレミスネットワーク全体管理
- セキュリティ担当: 全アセットのセキュリティ設定を包括管理

---
```

#### d. `invert` — 反転カラー

ダークグレーの背景に切り替わります。重要な警告や話題転換のアクセントに使用します。要素の構造は通常スライドと同じで、配色のみ反転します。

```markdown
---

<!-- _class: invert -->

# 組織的な問題で対応が遅れてしまう

以下のような組織に関わる事象で、対応が遅れる

- 対応基準が存在していない
- 対応に関する経営層承認に時間がかかっている

---
```
